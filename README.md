# Landing Page – Advogada Criminalista

Este projeto consiste em uma landing page para uma advogada criminalista fictícia, Laura Monteiro.  

Com o objetivo de compor meu portfólio de desenvolvimento front-end e design web, página foi desenvolvida com foco em layout, organização visual e boas práticas de estilização.

## Tecnologias utilizadas
- HTML5
- CSS3

*Este repositório refere-se a um projeto fictício. As imagens utilizadas foram retiradas de bancos de imagens e todo o conteúdo apresentado é fictício.*
